Subclassing Text class to make a bitmap text - unfinished

This tutorial has been created for a stackoveflow question where a user needed to do a text object in fabricJS that works with bitmap instead of fonts. The code example contains everything you need to know to start a subclass. Consider that subclassing is a sort of last resource when you can't obtain the needed functionality with other apis ( events, custom controls ).
