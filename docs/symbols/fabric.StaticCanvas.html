{String} toDataURL(options)
Exports canvas element to a dataurl image. Note that when multiplier is used, cropping is scaled appropriately 
Generate jpeg dataURL with lower quality
var dataURL = canvas.toDataURL({
  format: 'jpeg',
  quality: 0.8
});
Generate cropped png dataURL (clipping of canvas)
var dataURL = canvas.toDataURL({
  format: 'png',
  left: 100,
  top: 100,
  width: 200,
  height: 200
});
Generate double scaled png dataURL
var dataURL = canvas.toDataURL({
  format: 'png',
  multiplier: 2
});
Parameters:
{Object} options Optional
Options object
{String} options.format Optional, Default: png
The format of the output image. Either "jpeg" or "png"
{Number} options.quality Optional, Default: 1
Quality level (0..1). Only used for jpeg.
{Number} options.multiplier Optional, Default: 1
Multiplier to scale by
{Number} options.left Optional
Cropping left offset. Introduced in v1.2.14
{Number} options.top Optional
Cropping top offset. Introduced in v1.2.14
{Number} options.width Optional
Cropping width. Introduced in v1.2.14
{Number} options.height Optional
Cropping height. Introduced in v1.2.14
Returns:

{String} toJSON(propertiesToInclude)
Returns JSON representation of canvas 
JSON without additional properties
var json = canvas.toJSON();

var json = canvas.toJSON(['lockMovementX', 'lockMovementY', 'lockRotation', 'lockScalingX', 'lockScalingY', 'lockUniScaling']);

canvas.includeDefaultValues = false;
var json = canvas.toJSON();

{String} toSVG(options, reviver)
Returns SVG representation of canvas 
Normal SVG output
var svg = canvas.toSVG();
SVG output without preamble (without <?xml ../>)
var svg = canvas.toSVG({suppressPreamble: true});
SVG output with viewBox attribute
var svg = canvas.toSVG({
  viewBox: {
    x: 100,
    y: 100,
    width: 200,
    height: 300
  }
});
SVG output with different encoding (default: UTF-8)
var svg = canvas.toSVG({encoding: 'ISO-8859-1'});
Modify SVG output with reviver function
var svg = canvas.toSVG(null, function(svg) {
  return svg.replace('stroke-dasharray: ; stroke-linecap: butt; stroke-linejoin: miter; stroke-miterlimit: 10; ', '');
});
Parameters:
{Object} options Optional
Options object for SVG output
{Boolean} options.suppressPreamble Optional, Default: false
If true xml tag is not included

{String} options.encoding Optional, Default: UTF-8
Encoding of SVG output
{String} options.width Optional
desired width of svg with or without units
{String} options.height Optional
desired height of svg with or without units
